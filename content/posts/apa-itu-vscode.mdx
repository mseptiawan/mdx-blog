# Apa itu Visual Studio Code (VS Code)?

Jika kamu baru saja terjun ke dunia pemrograman, kemungkinan besar kamu akan sering mendengar nama **Visual Studio Code** atau biasa disingkat **VS Code**.

VS Code adalah sebuah _source-code editor_ gratis yang dikembangkan oleh Microsoft untuk Windows, Linux, dan macOS. Meskipun terlihat ringan, editor ini sangat bertenaga dan fleksibel.

<Callout type="info">
  **Penting:** Banyak orang antara Visual Studio dan Visual Studio Code. Visual
  Studio adalah IDE (Integrated Development Environment) yang sangat berat,
  sedangkan VS Code adalah editor yang jauh lebih ringan namun bisa dikembangkan
  fungsinya.
</Callout>

---

## Mengapa VS Code Sangat Populer?

Berdasarkan survei Stack Overflow, VS Code secara konsisten menempati peringkat pertama sebagai alat pengembangan yang paling banyak digunakan. Berikut adalah alasannya:

### 1. Gratis dan Open Source

Kamu tidak perlu membayar sepeser pun untuk menggunakan VS Code. Selain itu, basis kodenya (VS Code - OSS) bersifat open source, yang memungkinkan komunitas untuk terus berkontribusi dalam pengembangannya.

### 2. Performa Ringan dengan Electron

Meskipun dibangun di atas framework **Electron** (yang menggunakan teknologi web seperti JavaScript dan Chromium), VS Code berhasil mengoptimalkan penggunaan memorinya sehingga tetap terasa cepat dibandingkan editor berbasis Electron lainnya.

### 3. IntelliSense yang Cerdas

Ini bukan sekadar auto-complete biasa. IntelliSense memberikan saran berdasarkan tipe data, definisi fungsi, dan modul yang kamu impor.

```javascript
// Contoh IntelliSense akan tahu metode apa saja yang tersedia untuk array
const buah = ["Apel", "Mangga"];
buah.map((b) => b.toUpperCase());
```
